A simple markdown editor powered by topolr

## Usage:

### step 1.

install npm modules

```
$ npm install
```

### step 2.

install topolr-builter and built the project

```
$ npm install topolr-builter -g
```

### step 3.

run the project

```
$ npm run app
```

## build

### for mac

```
$ npm run build:mac
```

### for windows

```
$ npm run build:windows
```

### for linux

```
$ npm run build:linux
```
